Paquetes y suscripciones

¿Por qué S2Webs usa un modelo de suscripción?

Porque permite repartir el coste de una web profesional y mantenerla actualizada con mejoras, soporte y seguridad.

¿Qué incluye mi suscripción mensual?

Diseño, hosting, soporte, actualizaciones, seguridad y acceso a herramientas según el paquete contratado.

¿Por qué pago si mi web todavía no está publicada?

La suscripción empieza cuando comienza el trabajo de consultoría, planificación y creación de tu web.

¿Puedo cambiar de paquete?

Sí, puedes mejorar tu paquete cuando tu negocio necesite más funcionalidades.

¿Cómo cancelo mi web?

Si tienes algún problema, contacta con nosotros y buscaremos una solución o te indicaremos los siguientes pasos.

Dominios y hosting

¿Puedo alojar mi web con otro proveedor?

Las webs Agentic deben estar alojadas en la infraestructura preparada para la plataforma, lo que garantiza seguridad, rendimiento y acceso al editor.

¿S2Webs ofrece hosting?

Sí, el hosting fiable está incluido en nuestros paquetes.

¿S2Webs ofrece emails?

Podemos ayudarte a configurar emails profesionales asociados a tu dominio.

¿Cada cuánto se renueva el dominio?

Normalmente cada año. Podemos ayudarte con renovaciones y gestión para evitar interrupciones.

¿Qué es el hosting?

Es el servicio que permite que tu web esté disponible en internet.

¿Puedo conservar mi dominio actual?

Sí, podemos ayudarte a conectarlo o transferirlo sin dejar tu web fuera de servicio.

Diseño y desarrollo

¿Cuánto costará mi web?

Depende del paquete y necesidades del proyecto. Contacta con S2Webs para recibir orientación.

¿Seré propietario de mi web?

Tu diseño y contenido estarán creados para tu negocio. Las condiciones concretas dependen del paquete y contrato.

¿Podéis buscar imágenes para mi web?

Sí, podemos ayudarte a seleccionar imágenes adecuadas si no tienes material propio.

¿Podré editar mi web?

Sí, Agentic permite editar contenido de forma sencilla.

¿Necesito habilidades técnicas?

No. El editor está pensado para usuarios sin conocimientos técnicos.

¿Me enseñaréis a usar la web?

Sí, incluimos formación para que puedas gestionar tu contenido con confianza.

¿Mi web será responsive?

Sí, se diseñará para funcionar correctamente en móvil, tablet y ordenador.

¿Podéis rediseñar una web existente?

Sí, podemos actualizar o rediseñar tu web actual.

¿Cuánto tarda una web?

Depende del alcance, pero trabajamos para que el proceso sea ágil y claro.

SEO

¿Qué es SEO?

SEO es la optimización para buscadores. Ayuda a que más clientes encuentren tu web en Google.

¿Mi web aparecerá en Google?

Diseñamos las webs con una base SEO sólida para darles más posibilidades de aparecer en búsquedas relevantes.

¿Cómo puedo medir el rendimiento?

Podemos configurar herramientas como Google Analytics y Google Search Console.

¿Cuánto tarda posicionar?

Depende del sector, competencia y estrategia. El SEO requiere trabajo continuo y paciencia.
